# Synera: Synergy Auto-Arena (Qt/QML)

本项目是基于 Qt 6 + QML + C++ 的单机自走棋实现版本，覆盖 `problem.md` 中阶段一到阶段三的核心功能，并实现阶段四扩展功能（利息 + 连胜/连败经济）。

## 1. 目录结构

```text
.
├── CMakeLists.txt
├── qml/
│   └── Main.qml
└── src/
    ├── main.cpp
    ├── unit.h
    ├── unit.cpp
    ├── gamecontroller.h
    └── gamecontroller.cpp
```

## 2. 核心模块

- `GameController`：游戏主控制器，负责
  - 三阶段循环（Prep / Combat / Resolve）
  - 单位移动、索敌、攻击、施法、死亡判定
  - 敌人波次生成与随轮次增强
  - 商店、金币、人口上限升级
  - 羁绊结算
  - 升星（3合1）
  - 装备掉落与穿戴
  - 存档/读档（JSON）
- `UnitData`：统一单位数据模型，敌我都使用同一结构，通过 `owner` 区分
- QML `Main.qml`：完整 UI（棋盘、备战区、商店、装备池、羁绊、经济/轮次/阶段信息）

## 4. 构建与运行

```bash
cmake -S . -B build
cmake --build build
./build/appSynera
```

> 需要 Qt 6.5+（Core/Gui/Qml/Quick）

## 5. 操作说明

- 准备阶段可以从商店购买单位到 Bench。
- 将单位从 Bench 拖拽到棋盘下半区可上阵；可在 Bench/Board 间互换。
- 右上角按钮可手动开始战斗、刷新商店、升级人口。
- 战斗胜利后概率掉落装备；点击装备，再右键目标单位可穿戴。
- 支持保存与加载（默认 `savegame.json`）。

## 6. AI 使用说明
